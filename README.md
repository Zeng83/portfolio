# AI Forest Portfolio

A runnable Next.js + React + Tailwind portfolio starter based on the mockup design.

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Stack

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS

## Files

- `app/page.tsx` — main portfolio page
- `app/layout.tsx` — root layout and metadata
- `app/globals.css` — global styles and forest background
- `tailwind.config.ts` — Tailwind setup
- `postcss.config.js` — PostCSS setup

## Customize

Update these sections in `app/page.tsx`:

- `skills`
- `experiences`
- `projects`
- contact email
- hero text and titles
